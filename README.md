# Tienda de Ropa API (Backend)

Backend profesional para una tienda de ropa, construido con **Python + FastAPI + SQLAlchemy + PostgreSQL (Supabase) + Alembic**.

## Stack
- Python
- FastAPI
- SQLAlchemy
- PostgreSQL (Supabase)
- Alembic (migraciones)

## Requisitos
- Python 3.11+ (recomendado)
- PostgreSQL (local o Supabase)
